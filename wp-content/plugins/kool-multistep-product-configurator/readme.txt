== Description ==
  Easy multistep WooCommerce product configurator Plugin ,You can now define multistep in your products with the attributes and variations of your products.
  Documentation
  <ul>
<li>  Define Attributes<li>
<li>  Make product variable<li>
<li>  Checked "Multistep product configurator"<li>
<li>  Click on Attribute menu in product page , add your defined attributes here <li>
<li>  Assign variations to them </li>
<li>  Click on "KMSPC" for settings in product page </li>
</ul>   
 
<b>How does it work<b>

If you already using variable products with some variations, it only takes a few seconds to enable KMSPC for a single variable product.
Enable KMSPC for a variable product via Product Data Metabox.
Set some options like module, columns etc.
Thatâ€™s it! You improved the usability of a variable product!

<b>Features</b>

Supported modules: Steps, Accordion, Tabs, Vertical Steps
Set an attribute thumbnail
Horizontal and vertical Grid Item Layout
Set number of columns
Make summary to fullwidth
Reposition product image or hide it at all
Responsive
 
== Installation ==

1. Upload `Kool Multistep Product Configurator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
  <ul>
<li>  Define Attributes<li>
<li>  Make product variable<li>
<li>  Checked "Multistep product configurator"<li>
<li>  Click on Attribute menu in product page , add your defined attributes here <li>
<li>  Assign variations to them </li>
<li>  Click on "KMSPC" for settings </li>
</ul>
